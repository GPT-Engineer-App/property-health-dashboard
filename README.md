# property-health-dashboard

A business dashboard for building property owners, where they can see the "health status" of a building (rated from Red to Green, green being the best), based on various infrastructural parameters of the property such as water pressure leakage, fire and smoke hazard data, humidity data, temperature data, Co2 data

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/property-health-dashboard.git
cd property-health-dashboard
npm i
```


This will run a dev server with auto reloading and an instant preview.
